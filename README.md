 Volt ⚡

Volt is a beginner-friendly CubeSat telemetry system inspired by real satellite engineering.

The goal of this project is to learn how small satellites collect, process, and communicate telemetry data by designing a CubeSat from the ground up. The project includes a custom PCB, a 3D enclosure, and an ESP32-based control system with environmental sensors.

 Features

- ESP32-based telemetry system
- Temperature and light monitoring
- Custom PCB design
- 3D CubeSat enclosure
- Engineering documentation

 Tools

- EasyEDA
- Onshape
- GitHub
